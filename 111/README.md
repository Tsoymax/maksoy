# Maksoy 1.0

Мобильный веб-навигатор для Узбекистана на базе OpenStreetMap.

## Возможности

- определение местоположения и слежение за пользователем;
- поиск адресов и мест по Узбекистану;
- автомобильные маршруты, расстояние, время и ETA;
- пошаговые инструкции, голосовые подсказки и автоматическая перепрокладка;
- сохранённые места в памяти браузера;
- локальные отметки дорожных событий сроком на 24 часа;
- адаптивный интерфейс для телефона и компьютера.

## Запуск

Геолокация браузера надёжно работает на `localhost` или по HTTPS. Запустите папку через любой локальный веб-сервер, например:

```powershell
python -m http.server 8080
```

После этого откройте `http://localhost:8080` и разрешите доступ к геолокации.

## Важно

Версия использует публичные сервисы OpenStreetMap, Nominatim и OSRM. Для коммерческого запуска понадобятся собственный или договорной сервер маршрутизации/поиска, серверное хранение дорожных событий, аналитика и политика конфиденциальности.
